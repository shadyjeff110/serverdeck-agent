# ServerDeck Agent

The small, statically linked Linux management agent used by the ServerDeck macOS application.

ServerDeck downloads architecture-specific binaries from GitHub Releases, verifies their SHA-256 checksums, and installs them over an existing SSH connection.

Supported release assets:

- `serverdeck-agent-linux-amd64`
- `serverdeck-agent-linux-arm64`
- `SHA256SUMS`
